# debianConfig


### Update packages list and upgrade them
```bash
apt update && apt upgrade
```

### Install nodejs
```bash
apt install nodejs
```

### Install vim
```bash
apt install vim
```

### Install curl
```bash
apt install curl
```

### Install StringManolo's bash shell extension
```bash
curl 'https://raw.githubusercontent.com/StringManolo/SMBSE/main/.bashrc' -o ~/.basrc && source ~/.bashrc
```
